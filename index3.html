Binary Search

Search Algorithm
O(log n)

Quick Sort

Sorting Algorithm
O(n log n)

A highly efficient **divide-and-conquer** method that chooses a 'pivot' and moves smaller elements to the left, and larger to the right.

pivot = partition(arr, low, high)

Hash Table

Data Structure
O(1) Avg.

Maps keys to values using a special function (hash function) allowing for incredibly fast insertions and lookups.

index = hash(key) % capacity

Merge Sort

Sorting Algorithm
O(n log n)

Divides the list into two halves until single items remain, then merges the halves back together in sorted order.

merge(leftHalf, rightHalf)

Dijkstra's Algo

Graph Pathfinding
O(E + V log V)

Finds the **shortest path** from a starting node to all other nodes in a graph with positive edge weights. Used in navigation.

priorityQueue.add(node, distance)

BFS

Graph Traversal
O(V + E)

Explores a graph level by level, checking all neighbors before moving deeper. Uses a **Queue** data structure.

queue.enqueue(startNode)

Dynamic Programming

Optimization Technique
Varies

Method for solving complex problems by breaking them down into smaller subproblems and storing the solutions to avoid redundant work.

memo[i] = solve(i-1) + solve(i-2)

DFS

Graph Traversal
O(V + E)

Explores a graph by going as deep as possible down one branch before backtracking. Uses a **Stack** data structure.

dfs(visited, node.child)
